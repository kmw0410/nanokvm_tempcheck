# NanoKVM temprature check script

how to use
 - download this script using `wget`
    ```bash
    wget https://raw.githubusercontent.com/kmw0410/nanokvm_tempcheck/refs/heads/main/temp.sh
    ```

 - grant execute permissions with `chmod`
   ```bash
   chmod +x ./temp.sh
   ```
 - run this script
   ```bash
   # just once:
   ./temp.sh

   # using watch command:
   watch sh ./temp.sh
   ```
